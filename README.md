# Translate API

### Objective:

The objective of the API is to provide access to the Googletrans Python library for other languages that do not have such a good and easy-to-use solution, without the need for an API key or subscription.

### URL:
    https://translate-api-0unn.onrender.com/getTranslation/text|lang

#### Example:
    https://translate-api-0unn.onrender.com/getTranslation/Hello|pt
